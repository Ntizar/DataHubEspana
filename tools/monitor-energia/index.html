ℹ️ Sobre estos datos

Datos de ejemplo del sistema eléctrico español. Para datos reales, conecta el esios_scraper.py con la API de Red Eléctrica Española (ree.es). La demanda eléctrica de España ronda los 25.000-40.000 MW según la hora y temporada. Las renovables (eólica + solar + hidro) suelen cubrir el 40-60% de la demanda en días buenos.
